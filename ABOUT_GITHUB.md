An end-to-end data engineering and AI project processing over 1 Million Dubai real estate transactions. Built from scratch, this project features a robust Python ETL pipeline, a high-performance DuckDB data warehouse, and a custom LangChain Text-to-SQL agent powered by Groq (Llama-3). It demonstrates applied data science, rigorous prompt engineering, and the creation of an interactive Streamlit/Plotly BI dashboard.
